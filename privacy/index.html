Privacy Policy

Last updated 2026-05-06.

This policy describes how the application Nicohtown Auto-Draft ("the Application") handles data. The Application is operated by Schaefer Properties / Prominent Wealth Strategies LLC ("the Operator") for the sole purpose of staging draft content into the TikTok inbox of the @nicohtown account for manual review.

Data the Application Receives

When a TikTok account owner authorizes the Application via TikTok's OAuth flow, the Application receives:

  • An access token and refresh token bound to the authorizing TikTok account.
  • Basic profile information made available by the requested OAuth scope (e.g., open ID, display name).

The Application does not request or store TikTok passwords, payment information, contact lists, or private messages.

How Data is Used

The Application uses the access token only to call TikTok's Content Posting API on behalf of the authorizing account. Specifically, the Application uploads draft videos to the account's inbox so the account owner can review and publish them manually inside the TikTok app. No other API calls are made.

Data Storage

OAuth tokens are stored on infrastructure controlled by the Operator (a private server inside a residential network and the Operator's Railway-hosted dashboard). Tokens are encrypted at rest where storage permits and are accessible only to the Operator and authorized personnel. Tokens are not shared with third parties, sold, or used for analytics, advertising, or model training.

Retention & Revocation

Tokens are retained as long as the Application is in active use by the authorizing account. The account owner can revoke access at any time through TikTok's connected-apps settings, after which the Application will no longer be able to make API calls on the account's behalf. The Operator will delete stored tokens upon revocation or written request.

Children's Privacy

The Application is not directed at children under 13 and does not knowingly collect data about minors.

Changes

If this policy changes materially, the updated date at the top of the page will be revised. Because the Application is a single-user internal tool, changes are not separately notified.
